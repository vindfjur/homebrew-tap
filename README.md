# vindfjur/homebrew-tap

Homebrew formulae for my projects.

## Install

```bash
brew install vindfjur/tap/ephemdir
```

This taps the repository automatically. To tap explicitly:

```bash
brew tap vindfjur/tap
brew install ephemdir
```

## Formulae

| Formula    | Description                                                        |
| ---------- | ----------------------------------------------------------------- |
| `ephemdir` | Temporary directories that clean themselves up — after a lifetime or on restart. ([repo](https://github.com/vindfjur/ephemdir) · [PyPI](https://pypi.org/project/ephemdir/)) |
